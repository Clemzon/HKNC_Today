Menu Overview

This page is now organized in two simple parts. The weekly menu is meant to give students a clean, easy-to-read view of the current week. The monthly menu section provides access to the full official menu document from Microsoft OneDrive.

The menu may change based on campus operations, product availability, or kitchen updates. When in doubt, check the monthly menu document or ask dining staff for the most current information.

Important Notes

Menu items are subject to change. Please check back regularly for updates.

If a meal listed on this page does not match what is being served, please follow the dining hall’s posted information or ask staff for the latest update.

Staff Update Process

Staff should continue using the Microsoft workflow for menu publishing:

  1. Upload or replace the current menu document in OneDrive.
  2. Update the Weekly Menu list in Microsoft Lists.
  3. Confirm the published weekly information matches the official file.

This page is now structured to support that process without requiring staff to edit large amounts of HTML content.
